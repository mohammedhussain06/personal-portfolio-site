# Mohammed Hussain Feroz Shaikh — Personal Portfolio

> A modern, dark-themed personal portfolio website built with vanilla HTML, CSS & JavaScript.

🌐 **Live Site:** [mohammedhussain06.github.io/portfolio-website](https://mohammedhussain06.github.io/portfolio-website) *(update after deploying)*

---

## ✨ Features

- **Animated Neural Network Canvas** — interactive particle background that reacts to mouse movement
- **Custom Cursor** — smooth dual-cursor effect with hover interactions
- **Typed Text Effect** — cycling role titles in the hero section
- **Scroll Reveal Animations** — elements fade in as you scroll
- **3D Tilt Effect** — project cards tilt on mouse hover
- **Glassmorphism Design** — dark AI/tech aesthetic with glassmorphism cards
- **Fully Responsive** — works on mobile, tablet, and desktop
- **Sticky Navbar** — with active section highlighting
- **Contact Form** — with client-side submit feedback
- **All Live Links** — projects, GitHub, LinkedIn, and achievement certificates

## 🚀 Running Locally

No installation needed. Just open the file:

```bash
# Option 1 — Double-click index.html in File Explorer

# Option 2 — Python simple server
python -m http.server 8000
# Open http://localhost:8000

# Option 3 — VS Code Live Server extension
# Right-click index.html → "Open with Live Server"
```

## 🌍 Deployment (GitHub Pages)

1. **Create a GitHub repo** (e.g. `portfolio-website`)

2. **Push the code:**
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio"
   git remote add origin https://github.com/mohammedhussain06/portfolio-website.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to repo → **Settings** → **Pages**
   - Source: **Deploy from branch** → `main` → `/ (root)`
   - Click **Save**

4. **Live at:**
   ```
   https://mohammedhussain06.github.io/portfolio-website
   ```
